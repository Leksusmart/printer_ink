Для начала установки скачайте репозиторий, предванительно открыв терминал в любой директории и выполнив команду:
```bash
git clone https://github.com/Leksusmart/printer_ink.git
```

## №1. Инструкция по запуску проекта без Docker

### 1.1 Установка всех зависимостей
Для запуска проекта на вашем компьютере должен быть установлен **Node.js** (рекомендуемая версия LTS(22)).
Для работы проекта на вашем компьютере требуется иметь **PostgreSQL** с готовыми таблицами и данными. Для этого можно использовать восстановление БД с помощью файла `init.sql`, выбрав Format - **Plain**.
Для настройки проекта нужно разместить файл с именем `.env` в корневую папку и заполнить её своими данными(Пример в файле `.env.example`).

Далее откройте терминал в корневой папке проекта `printer_ink` и выполните команду, которая автоматически установит все библиотеки для корня, бэкенда и фронтенда:
```bash
npm run init-all
```

### 1.2 Запуск проекта
После успешной установки запустите бэкенд и фронтенд одной командой:
```bash
npm start
```

## №2. Инструкция по запуску проекта при помощи Docker

Для запуска проекта на вашем компьютере должен быть установлен **Docker Desktop**.

Чтобы запустить проект в Docker, нужно выполнить команды в корневой директории проекта:

```bash
docker compose down -v
docker compose build
docker compose up
```

## №3. Инструкция для создания APK файла клиента

### 1.1 Установка всех зависимостей
Для запуска проекта на вашем компьютере должен быть установлен **Node.js** (рекомендуемая версия LTS(22)).
Для компилирования проекта на вашем компьютере должен быть **Android Studio**. 

Далее откройте терминал в корневой папке проекта `printer_ink` и выполните команду, которая автоматически установит все библиотеки для корня, бэкенда и фронтенда:
```bash
npm run init-APK
```

### 1.2 Запуск проекта
После успешной установки запустите обновление файлов проекта командой:
```bash
npm run updateAPK
```

Затем запустите проект в **Android Studio** с помощью команды:
```bash
npm run createAPK
```

После этого должна открыться Android Studio и начаться установка зависимостей и индексирование проекта,
подождите, затем соберите приложение через Android Studio
