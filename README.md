## Инструкция по запуску проекта

Для запуска проекта на вашем компьютере должен быть установлен **Node.js** (рекомендуемая версия LTS).

### 1. Установка всех зависимостей
Откройте терминал в корневой папке проекта (`printer_ink`) и выполните команду, которая автоматически установит все библиотеки для корня, бэкенда и фронтенда:
```bash
npm run init-all
```

### 2. Запуск проекта
После успешной установки запустите бэкенд и фронтенд одной командой:
```bash
npm start
```
* Бэкенд (NestJS) запустится на: `http://localhost:3000`
* Фронтенд (Next.js) запустится на: `http://localhost:3001`
